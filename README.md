## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
# choose one of the commands
$ npm install
$ yarn
```

## Running the app

```bash
# choose one of the commands
# development
$ npm run start
$ yarn start

# choose one of the commands
# watch mode
$ npm run start:dev
$ yarn start:dev

# choose one of the commands
# production mode
$ npm run start:prod
$ yarn start:prod
```

## Migrations
```bash
# Create a migration
$ npm run migration:create --name=foo

# Generate a migration from schema changes
$ npm run migration:generate --name=bar

# Run migrations and checks for schema changes
$ npm run migration:run

# Revert migrations
$ npm run migration:revert
```

## Docker
```bash
# Running all containers
$ docker-compose up -d

# Stopping all containers
$ docker-compose down
```

## Test

```bash
# choose one of the commands
# unit tests
$ npm run test
$ yarn test

# choose one of the commands
# e2e tests
$ npm run test:e2e
$ yarn test:e2e

# choose one of the commands
# test coverage
$ npm run test:cov
$ yarn test:cov
```
